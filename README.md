# AirPods Charging Case

## Project Description
This project is a 3D model of an AirPods charging case designed using Onshape. The dimensions were estimated from reference images, and the model includes the main external features such as:
- Rounded edges
- Lid separation line
- Front LED indicator
- Charging port
- Rear hinge
